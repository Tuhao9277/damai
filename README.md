# 💕damai
- 💝 纯前端实现的大麦网购票网站，利用gulp进行项目的基本构建
- ✨ 数据接口基本来自原站，通过ajax请求获取数据并渲染页面
### 🎉安装与下载预览
- 下载地址 
- 安装
`npm install`
